# Hot QCD equation of state

The file `hrg_hotqcd_eos_binary.txt` is a conversion from binary to text of the file `MUSIC/EOS/hotQCD/hrg_hotqcd_eos_binary.dat` coming from the MUSIC community github `MUSIC-fluid`.  
The files `p.txt`, `s.txt`, `T.txt` contains the respective pressure, entropy density, and temperature as function of the energy density (energy density being the first column).
